# Changelog

## [v1.0.0] - 10 novembre 2025

### Release iniziale: prima versione stabile

Questa è la prima release ufficiale dell’applicazione Travel Incentive App.
Include tutte le funzionalità di base per la gestione di viaggi incentive aziendali.

#### Funzionalità principali
- Autenticazione utenti (login, gestione profili)
- Visualizzazione agenda eventi e viaggi
- Gestione documenti di viaggio (upload, download)
- Galleria fotografica
- Pagina di esplorazione e informazioni di viaggio
- Modulo di registrazione partecipanti
- Gestione impostazioni utente
- Backend Node.js con API REST e mock data
- Test automatici su modelli e API

#### Miglioramenti e note tecniche
- UI responsive con Tailwind CSS
- Struttura modulare con React e TypeScript
- Scripts per gestione e popolamento database mock
- Configurazione pronta per deployment e CI/CD

#### Prossimi passi
- Raccolta feedback dagli utenti
- Ottimizzazione performance e sicurezza
- Pianificazione nuove feature (notifiche, analytics, multilingua)
